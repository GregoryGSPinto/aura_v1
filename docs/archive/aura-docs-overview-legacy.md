# Aura Docs

## Camadas

- Client Layer: frontend Next.js responsivo e pronto para PWA
- API Layer: FastAPI com endpoints versionados
- Core Layer: servicos de LLM, projetos, auth, memoria e comandos
- System Layer: macOS, terminal, VS Code, Ollama
