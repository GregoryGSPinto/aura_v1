# Memory Experience

Memoria deve ser util, editavel, governada e transparente.

## Superficies

- sinais de memoria na home
- memoria ativa no painel contextual
- pagina dedicada de memoria

## Regras de UX

- mostrar apenas memorias com utilidade operacional
- distinguir preferencia, contexto recente e memoria operacional
- permitir que o usuario entenda porque a memoria apareceu
